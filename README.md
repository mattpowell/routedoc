# routedoc &mdash; _documentation for routes._
`routedoc` is a tool for documenting and generating routes.

Example
===
```
/**
 * Api endpoint for searching the database
 * @name api-search
 * @param {string} q - text to query for.
 * @returns {SearchResultsJson}
 */
GET /search
```
